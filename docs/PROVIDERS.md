# Providers

Supported providers in v1.1.0:

- DeepSeek: balance query supported.
- SiliconFlow: balance query supported.
- Moonshot / Kimi: balance query supported.
- OpenAI: balance query not supported; local usage estimation supported.
- Anthropic / Claude: balance query not supported; local usage estimation supported.
- Zhipu GLM: balance query not supported; local usage estimation supported.

Provider balance APIs may change. If a provider changes its response format, balance display may require an update.
