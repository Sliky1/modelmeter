# Privacy Policy

ModelMeter is a local VS Code extension for monitoring AI provider balances and estimating local usage costs.

## What is stored locally

- API account aliases.
- Provider selections.
- Local 30-day usage history.
- API keys via VS Code SecretStorage.

## What is sent over the network

Only provider balance or connection-test requests are sent, and only to the selected provider API endpoint. API keys are used as authorization headers for those provider requests.

## Telemetry

ModelMeter does not collect telemetry and does not upload local usage history.
